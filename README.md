<h1 align="center" style="width: 50vw; text-align:center;">yl.service</h1>

<p align="center" style="width: 50vw">A Vue.js 3 UI library</p>

- 💪 Vue 3 Composition API
- 🔥 Written in TypeScript

## 快速开始
```
npm install
npm run dev
```

## 组件快速开发

```
# 命令式新增组件
npm run com:create

# 命令式删除组件
npm run com:clean
```

## 构建

```
# 代码打包
npm run build

# 构建docs实时交互文档库
npm run build:doc

# 构建lib组件库
npm run build:lib
```

## 代码格式化

```
# eslint 检查
npm run lint

# prettier 自动格式化
npm run prettier
```
