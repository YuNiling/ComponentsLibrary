# API

## 属性

|  参数   |      说明      |  类型   | 可选值 | 默认值 | 是否必填 |
| :-----: | :------------: | :-----: | :----: | :----: | :------: |
| `type`  |      类型      | string  |   -    |   -    |    否    |
| `plain` | 是否为朴素按钮 | boolean |   -    |   -    |    否    |
| `round` | 是否为圆角按钮 | boolean |   -    |   -    |    否    |
